## Content Rules
- Never use words on the banned list (see anti-slop.md)
- Short paragraphs. One idea per paragraph.
- Lead with the insight, not the setup.
- Specific > generic. Numbers > adjectives. Stories > claims.
- End with a thought, not a CTA (unless explicitly asked for one).
- No em dashes unless they're in my voice profile.
- No "I" in the first sentence of a LinkedIn post.

## Workflow
1. User describes the topic or shares raw material
2. Load voice profile (all 3 files)
3. Draft in my voice
4. Self-check against anti-slop rules
5. Format for the target platform
6. Present for review

## Platform Formats

### LinkedIn
- Hook in first line (visible before "see more")
- Short paragraphs (1-3 sentences)
- No links in post body (kills reach)
- Link in first comment if needed

### Blog
- 800-1500 words
- Clear H2 sections
- One actionable takeaway per section
- Code blocks if showing technical content

### Email Newsletter
- Conversational tone
- One topic per email
- Clear single CTA
- PS line for secondary link

### Twitter/X Thread
- First tweet is the hook
- One idea per tweet
- No hashtags in thread body
- Last tweet: summary + CTA
