# Anti-Slop Rules

AI-generated content has tells. These rules eliminate them. Load this file alongside `core-voice.md` before writing any content.

---

## Banned Words and Phrases

### Corporate Buzzwords
| Instead of | Write |
|-----------|-------|
| leverage | use |
| utilize | use |
| optimize | improve / fix |
| streamline | simplify |
| synergy | [delete the sentence] |
| paradigm shift | [say what actually changed] |
| game-changer | [say what it actually changes] |
| best-in-class | [say why it's better, specifically] |
| cutting-edge | [say what's new about it] |
| robust | [say what makes it strong] |
| scalable solution | [say what scales and how] |
| holistic approach | [say what you actually do] |
| move the needle | [say what metric improves] |

### AI Slop Tells
| Pattern | Fix |
|---------|-----|
| "In today's [landscape/world/era]..." | Delete. Start with the point. |
| "It's worth noting that..." | Delete. Just note it. |
| "Let me be clear..." | Delete. Just be clear. |
| "At the end of the day..." | Delete. Say the thing. |
| "The reality is..." | Delete. State the reality. |
| "Here's the thing..." | Delete. Say the thing. |
| "Look," / "Listen," | Delete unless it's genuinely your voice. |
| Starting with "So," | Delete the "So,". |
| "This is a great question" | Delete. Answer the question. |

### Punctuation Tells
| Pattern | Fix |
|---------|-----|
| Em dashes everywhere (—) | Use only if it's in your actual voice profile. Max 1 per post. |
| Semicolons in casual writing | Use a period instead. |
| Exclamation marks (!) | Max 1 per piece. Earn it. |
| Ellipsis (...) for drama | Use sparingly or not at all. |

### Structure Tells
| Pattern | Fix |
|---------|-----|
| 3-item lists everywhere | Vary your structure. |
| "First... Second... Third..." | Mix up transitions. |
| Every paragraph is exactly 2 sentences | Vary paragraph length. |
| Ending every post with a question | Questions are fine. Not every time. |

## The Specificity Test

Before publishing, check every claim:
- **Vague:** "This saved us a lot of time"
- **Specific:** "This cut our list building from 4 hours to 20 minutes"

- **Vague:** "Many companies struggle with this"
- **Specific:** "3 of the 5 Series B SaaS companies I talked to last month had this problem"

If you can't be specific, either research it or cut it.

## The Read-Aloud Test

Read the content out loud. If any sentence makes you cringe or sounds like a press release, rewrite it in the words you'd actually use talking to a friend in the industry.

---

*These rules are non-negotiable. Every piece of content gets checked against this list.*
