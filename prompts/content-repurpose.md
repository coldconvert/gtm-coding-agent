# Content Repurpose Prompt

Use this prompt to take one piece of content and turn it into 5 formats. One idea, five outputs, consistent voice.

---

## The Prompt

```
You are a content strategist helping me repurpose a single piece of content into 5 formats. Before generating, I need to give you two things:

### Input 1: The Source Content

Paste the original content below. This can be a blog post, podcast transcript, talk, newsletter, internal doc, customer conversation — anything with substance.

[PASTE SOURCE CONTENT HERE]

### Input 2: Voice Profile

If you have a voice DNA profile (from templates/voice/profile.md), paste it here. If not, answer these questions:

1. How formal/casual is your writing? (Scale: corporate → professional → conversational → casual)
2. Do you use "I" or "we"?
3. Any phrases, patterns, or stylistic habits you're known for?
4. What do you NEVER say? (Buzzwords, cliches, phrases that make you cringe)
5. Who is a writer or creator whose tone is similar to yours?

---

Now generate all 5 formats below. Every format must:
- Maintain the same core insight/argument from the source
- Match the voice profile exactly
- Stand alone — someone reading just one format should get the full value
- NOT sound like they were generated by AI. No "In today's fast-paced world..." No "Let's dive in." No emoji unless the voice profile uses them.

---

## Format 1: LinkedIn Post

- Length: 800-1,200 characters (roughly 150-200 words)
- Structure: Hook (first line must stop the scroll) → Insight → Supporting point or story → Takeaway
- First line is everything. It appears before the "see more" fold. Make it earn the click.
- No hashtags in the body. If you must, 3 max at the very end.
- Line breaks between every 1-2 sentences. LinkedIn is read on phones.
- End with a perspective, not a question begging for engagement. "Agree?" is not a CTA.

## Format 2: Email Newsletter Section

- Length: 150-250 words
- Structure: One key insight from the source content, written as a section within a larger newsletter
- Assumes the reader has context on your space but hasn't seen the source content
- Include a link placeholder: [LINK TO FULL PIECE]
- Conversational but dense — every sentence should carry weight
- End with a takeaway the reader can use today

## Format 3: Twitter/X Thread

- 4-6 tweets
- Tweet 1: The hook. State the insight in a way that makes people want the full thread.
- Tweets 2-4: The supporting points. One idea per tweet. Concrete, specific, no filler.
- Tweet 5-6: The takeaway and (optionally) a link to the full piece.
- Each tweet must stand alone if someone sees it in isolation.
- No "Thread 🧵" opener. No numbered tweets. No "1/" prefix.
- Character limit per tweet: 280

## Format 4: Blog Post Outline

- Not the full blog post — just the outline someone could write from (or hand to a writer)
- Structure: Title → Subtitle → 4-6 section headers with 2-3 bullet points each describing what goes in that section
- Opening section: the hook and why this matters now
- Closing section: the actionable takeaway
- Estimated word count: 1,000-1,500 words for the final post
- SEO note: suggest 2-3 keywords to target based on the content

## Format 5: Short-Form Video Script (60-90 seconds)

- Format: spoken word, as if talking to camera or recording a Loom
- Structure: Hook (first 3 seconds) → Setup (what's the problem/observation) → Insight → Proof or example → CTA
- Write it as spoken language, not written language. Contractions. Short sentences. The way you actually talk.
- Include [VISUAL] cues where a screen share, graphic, or text overlay would help
- First line must hook within 3 seconds. "Here's something nobody talks about..." or jump straight into the insight.
- End with one clear CTA: follow, check out the link, try this yourself.

---

For each format, also include:
- **Best time to publish:** [recommendation]
- **One thing to customize before posting:** [what to tweak for the specific platform/audience]
```

## How to Use

1. Start with your highest-value content — the piece that got the most engagement, the talk that landed best, the insight customers keep referencing.
2. If you have a voice DNA profile in `templates/voice/profile.md`, reference it. The voice consistency is what makes repurposed content feel intentional instead of recycled.
3. Paste the prompt with your source content and voice profile.
4. Review each format individually. The AI handles structure and adaptation. You handle voice accuracy, factual correctness, and the judgment calls.
5. Publish across platforms over 1-2 weeks. Don't dump all 5 on the same day.

## Repurposing Calendar

A simple cadence that works:
- **Monday:** LinkedIn post
- **Tuesday:** Twitter/X thread
- **Wednesday:** Send newsletter
- **Thursday:** Publish blog post
- **Friday:** Record and post video

One source piece fuels an entire week of content across 5 channels.
