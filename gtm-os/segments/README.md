# Segments

Each file in this folder represents a target account segment.

## How to Create a Segment

Create a markdown file named `segment-name.md` with:

```markdown
# Segment: [Name]

## Criteria
- Industry:
- Company size:
- Geography:
- Tech stack:
- Additional filters:

## Size
- Total addressable: [count]
- Currently in CRM: [count]
- Active in sequences: [count]

## Status
- [ ] List built
- [ ] Enriched
- [ ] Sequenced
- [ ] Results reviewed

## Performance
| Metric | Value |
|--------|-------|
| Emails sent | |
| Open rate | |
| Reply rate | |
| Meetings booked | |

## Notes
[Observations, adjustments, what's working]
```

## Naming Convention
Use descriptive names: `series-b-saas-east-coast.md`, `enterprise-fintech-uk.md`
