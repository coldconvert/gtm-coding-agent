# Engine Prompts

Reusable AI prompts for GTM operations. These are the prompts your coding agent uses when running GTM tasks.

## How to Use
Reference these in your CLAUDE.md or call them directly:
- "Use the qualification prompt on [company]"
- "Run the account research prompt for [domain]"

## Available Prompts
See `../../prompts/` for the full library of ready-to-use prompts.

## Adding Custom Prompts
Create a new `.md` file here with:
1. A clear title and purpose
2. Input variables (what the user needs to provide)
3. The prompt itself
4. Expected output format
