# Active Campaigns

Each active campaign gets its own markdown file here.

## Campaign File Template

```markdown
# Campaign: [Name]

## Overview
- **Type:** [Outbound / Content / ABM / Paid / Event]
- **Start date:** [YYYY-MM-DD]
- **Target segment:** [Link to segment file]
- **Attack angle:** [Which angle from messaging/attack-angles.md]
- **Owner:** [Who's running this]

## Setup
- **List size:** [Number of accounts/contacts]
- **Sequence:** [Number of steps, channels used]
- **Content:** [What content is being used]

## Metrics (Updated Weekly)
| Week | Sent | Opens | Replies | Meetings | Notes |
|------|------|-------|---------|----------|-------|
| W1 | | | | | |
| W2 | | | | | |

## Decisions
- [Date]: [What was changed and why]

## Status
- [ ] List built
- [ ] Sequence written
- [ ] Launched
- [ ] Week 1 review
- [ ] Week 2 review
- [ ] Decision: scale / iterate / kill
```
