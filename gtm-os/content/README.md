# Content Pipeline

Track your content production here.

## Structure
- Put content briefs and drafts in this folder
- Use `../../templates/content/` for formats (blog, content drop, SEO brief)
- Load voice profile from `../../templates/voice/` before writing

## Suggested Organization
```
content/
├── briefs/          # Content briefs and SEO briefs
├── drafts/          # Work in progress
├── published/       # Final versions with publish dates
└── ideas.md         # Content idea backlog
```
